  • Seite neu laden

  • Blockierungssoftware deaktivieren oder Ausnahme-Regel erstellen
HTTPS / SSL / TLS Zertifikatsfehler
  • Ursache: temporäre DDoS-Sperre oder Virenscanner-Filter
  • Anderen Browser testen
  • Tipps zu „schwarzem Bild" oben befolgen
  • Router neu starten oder später erneut testen
  • DNS-Server ändern (Anleitung auf Website Serien.Domains)
Meldung: „CUII-Sperre"
  • Website Serien.Domains besuchen
  • Schritt-für-Schritt-Anleitung befolgen

  • Neuer Video-Link in 12–24 Std. verfügbar

  • VPN deaktivieren (nicht erforderlich)
  • Ggf. Anleitung zur CUII-Sperre befolgen

  • Chrome oder Edge verwenden
  • Firefox „Enhanced Tracking Protection" verursacht häufig Probleme

  • Videos werden noch umgewandelt/optimiert
  • Der Vorgang kann wenige Minuten bis Stunden dauern
  • Video-Link nach 1 Tag als defekt melden
Website kaputt (Funktionen gehen nicht)
  • anderen Browser testen, z.B. Google Chrome
  • Browser einmal schließen und erneut starten
  • Browser und Cache leeren oder Website im Incognito-Fenster öffnen.
